# 🌦️ WeatherNow

A simple weather application built with React and OpenWeather API that provides real-time weather information for any city.

## 🚀 Live Demo

https://maheshthammappa.github.io/weathernow

## ✨ Features

* Search weather by city name
* Real-time weather data
* Temperature display
* Humidity information
* Wind speed details
* Atmospheric pressure
* Feels-like temperature
* Responsive design

## 🛠️ Tech Stack

* React
* Vite
* OpenWeather API
* GitHub Actions
* GitHub Pages

## 📦 Installation

```bash
git clone https://github.com/maheshthammappa/weathernow.git
cd weathernow
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_KEY=YOUR_OPENWEATHER_API_KEY
```

## 🏗️ Build

```bash
npm run build
```
